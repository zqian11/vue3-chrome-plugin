# db-answer-tool

> 基于主分支模板开发，用于在题库已知的前提下，通过脚本辅助进行答题。

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Hot reload for chrome-plugin
```
yarn watch
```
